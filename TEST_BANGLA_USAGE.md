# Test Bangla Output - Usage Guide

## Overview
The `test_bangla_output.py` script allows you to preview Bangla summaries and titles without posting to Facebook. This helps you verify the quality of the generated text before the bot actually posts.

## Basic Usage

### Simple Test (Default)
```bash
python test_bangla_output.py
```
- Shows first 5 articles
- Displays full summaries
- Shows statistics
- Prints to console

### Save Output to File
```bash
python test_bangla_output.py --save
```
- Saves output to `bangla_test_output.txt`
- Also displays on console

### Custom Output File
```bash
python test_bangla_output.py --save --output my_test.txt
```

## Command Line Options

### `--save` or `-s`
Save output to a file (default: `bangla_test_output.txt`)

### `--output OUTPUT` or `-o OUTPUT`
Specify custom output file name

### `--no-stats`
Hide statistics section (faster output, less detail)

### `--truncate`
Show only first 200 characters of summaries (useful for quick previews)

### `--limit LIMIT` or `-l LIMIT`
Limit number of articles to show (default: 5)

## Examples

### Quick Preview (2 articles, no stats)
```bash
python test_bangla_output.py --limit 2 --no-stats
```

### Full Test with File Save
```bash
python test_bangla_output.py --save --output full_test.txt
```

### Quick Check (truncated summaries)
```bash
python test_bangla_output.py --truncate --limit 3
```

### Comprehensive Test
```bash
python test_bangla_output.py --save --output comprehensive_test.txt --limit 10
```

## What the Script Shows

For each article:
- ✅ Original Swedish title
- ✅ Translated Bangla title
- ✅ Bangla summary (generated by OpenAI)
- ✅ Character counts and text quality metrics
- ✅ Preview of what would be posted to Facebook
- ✅ Category and URL information

Overall statistics:
- ✅ Total articles scraped
- ✅ Average summary length
- ✅ Average Bangla character percentage
- ✅ Average words and sentences per summary

## Text Quality Metrics

The script analyzes:
- **Summary length**: Total characters
- **Bangla characters**: Count and percentage of Bangla Unicode characters
- **Word count**: Number of words
- **Sentence count**: Number of sentences (based on punctuation)

## Tips

1. **Check Bangla Character Percentage**: Should be high (70%+) for good quality
2. **Review Summary Readability**: Make sure summaries are natural and engaging
3. **Verify Title Translation**: Ensure titles are properly translated
4. **Check Length**: Summaries should be appropriate length for Facebook posts
5. **Save Results**: Use `--save` to keep a record of test runs

## Output File Format

When saved to file, the output includes:
- Test date and time
- Configuration information
- Detailed statistics
- Full article previews with all metrics
- Tips and recommendations

## Notes

- The script does NOT post to Facebook
- No Facebook credentials required
- Uses the same scraping and summarization as the actual bot
- OpenAI API key required (same as bot)
- Can be run anytime to test quality
